PIET-Alumni-Portal - Frontend bundle
-----------------------------------
Files:
- index.html, login.html, register.html, alumni.html, profile.html, jobs.html, groups.html, resources.html
- modern.css (theme)
- piet.png is NOT included. Add your college logo image named 'piet.png' in the same folder for best results.

How to use:
1. Download and unzip the package.
2. Open index.html in your browser.
3. Add piet.png image to the folder for the logo.
4. Hook up to a backend (Firebase/Node) later if needed.
